Exercise Summary:

This exercise consists of designing a webpage that is responsive in design, flexible for up to three types of screens, computer, tablet, and mobile. Realistically I know that we should apply diplay: block; to make all information available but in this case I am imaginging ads or filler on both sides that are not useful to the user hypothetically.

  • Computer Screen:
    • All columns visible
    • Navigation buttons moderate length
  • Tablet
    • Middle columns visible only
  • Mobile
    • Navigation buttons expanded to fill length

    • Basis

    Designing a responsive page that scales for mobile, tablet, and computer screens.

    Resize the browser window to see the responsive effect.
