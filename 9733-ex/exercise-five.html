Exercise Summary:

In this exercise we experiment with modifying several parameters to create an original design using the existing W3.css framework and then modifying it for personalization.

Specifically the following modifications have been done:

  • colour alterations
  • navigation adjustments
    • *in addition to a nested dropdown list
  • minor adjustments to layout
  • altered typography
  • addition of a header with a logo and tagline
  • addition of a footer, with a redundant text-only nav and a back-to-top link that works.
